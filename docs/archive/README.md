# Archived docs — build era

Historical documents from before the React cutover (PR #61) and the v1.0.0
release. They are **kept for reference only** — nothing in here describes the
current application, and nothing links to them as live guidance.

| File | What it was |
| --- | --- |
| `HANDOFF.md` | Session-memory handoff for the **vanilla JS** portal (pre-rebuild). |
| `PHASE2-HANDOFF.md` | Mid-rebuild handoff for the `react-rebuild` branch. |
| `ROADMAP.md` | Wave-by-wave build spec for the vanilla portal's feature waves. |
| `REACT-PARITY.md` | The parity checklist that gated the React cutover — all boxes checked. |
| `BACKLOG.md` | Vanilla-era feature backlog — everything shipped or superseded. |
| `superpowers/` | Working plans/specs from the June 2026 case-files build. |

Current documentation lives one level up: `docs/HANDBOOK.md` and
`docs/handbook/` (developer handbook), `docs/USER-GUIDE.md` (member guide),
`docs/HARDENING.md` (security checklist status), `docs/DEFERRED.md` (parked
work with triggers), `docs/RELEASE-READINESS.md` (v1.0.0 verification), and
the Owner Portal's in-app improvement roadmap
(`src/components/owner/ownerData.ts`).
